# Projeto-Imersao-Dev-Alura-Html-Css-Gemini
Este repositório foi criado com o objetivo de hospedar o desafio de projeto da Imersão Dev em Html, Css e Gemini da Alura. Em vez de criar um simples projeto, decidi construir um mundo imersivo e gamificado de aprendizado.
